## üçé English Instructions

Adaptive Yablochny chat preset and extension for SillyTavern.

### 1. Installation & Update
*   **Installation**:
    1.  Open SillyTavern, go to **Extensions** (Cubes icon) -> **Install Extension**.
    2.  Paste the repository URL and click **Install**.
    3.  **Important**: Reload the page after installation.
    4.  Go to the **Generation Parameters** tab (leftmost icon, Sliders). In the **Chat Completion Presets** section, select **"üçéYablochny Preset"**.
*   **Update**:
    1.  Manage Extensions -> Find **Yablochny Preset** -> Click **Update**.
    2.  **Important**: Reload the page.
    3.  Go to the extension settings (Cubes menu) and click **Sync preset**. This updates the prompt text while preserving your toggles and custom prompts.

### 2. Synchronization (Sync) Logic
The **Sync preset** button updates the preset's prompt content to the latest version.
*   **Preserved**: Your toggle states (on/off), custom-added prompts, and custom variant texts (if "Custom" is selected in settings).
*   **Updated**: Standard prompt content (e.g., system instructions) will be overwritten with the new versions from the extension.
*   **Reset**: If the preset breaks, delete it (trash icon in the Sliders tab) and click **Sync preset** in the extension.

### 3. Formatting (Butterfly Tab)
Manage formatting, Story String, and Reasoning in the **Formatting** tab (3rd icon from left, Butterfly).
*   **Gemini / Claude**: Set the **Start Reply With** (Prefill) field to `<think>`. This enables the reasoning mode for these models.
*   **GPT-4 / o1**: Keep the **Start Reply With** field **empty**. Filling it may lead to errors or hallucinations.

### 4. Preset Symbols (Butterfly Tab)
*   **‚óà (System)**: Technical core logic. Do not disable.
*   **‚Üó / ‚Üò / ‚Üí (Option)**: Exclusive options. Enable only one in a group.
*   **‚îå / ‚îî (Connection)**: Linked prompts. Enable the top one for the bottom one to function.
*   **‚ó¶ (Personal)**: Optional stylistic modifiers.
