## English Instructions

Adaptive Yablochny chat preset and extension for SillyTavern.

### 1. Installation & Update
*   **Install**: Extensions (Cubes) -> Install Extension -> Paste URL. **Reload the page**. Select "üçéYablochny Preset" in the **Generation Parameters** (Sliders tab, leftmost).
*   **Update**: Manage Extensions -> Update -> **Reload the page**. Click **Sync preset** in the extension settings to update the prompt content while keeping your toggles and custom prompts.

### 2. Formatting (Butterfly Tab)
Inside the **Butterfly tab** (3rd from left), you can find **Reasoning** and **Prefill** settings.
*   **Gemini / Claude**: Set **Start Reply With** to `<think>` to enable reasoning.
*   **GPT-4 / o1**: Keep **Start Reply With** **empty** to avoid errors.

### 3. Sync Logic
The **Sync preset** button updates standard prompt texts. Your **on/off states** and **manually added prompts** are preserved. If the preset breaks, delete it from the Sliders tab and click Sync again.
