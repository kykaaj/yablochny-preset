## üçé English Instructions

Adaptive Yablochny chat preset and extension for SillyTavern. This extension manages a Chat Completion preset that adapts to your settings and preserves your modifications.

### 1. Installation & Setup
1.  **Install Extension**:
    *   Open SillyTavern, go to **Extensions** (Cubes icon) -> **Install Extension**.
    *   Paste the repository URL and click **Install**.
    *   **CRITICAL**: Reload the SillyTavern page (F5) after installation.
2.  **Select the Preset**:
    *   Go to the **Generation Parameters** tab (leftmost icon, Sliders).
    *   In the **Chat Completion Presets** section, select **"üçéYablochny Preset"** from the list.

### 2. Updating
1.  Go to **Extensions** -> **Manage Extensions**.
2.  Find **Yablochny Preset** and click **Update**.
3.  **IMPORTANT**: Reload the page.
4.  Open the extension settings and click **Sync preset**. This updates the technical prompt content from GitHub while keeping your custom settings, toggles, and manual changes.

### 3. Synchronization (Sync) Logic
The **Sync preset** button is the "Update" button for the preset content.
*   **What is Preserved**:
    *   **Toggle states**: If you turned a specific prompt on/off in the preset, it stays that way.
    *   **Custom Prompts**: Any prompts you added manually to the preset list are kept.
    *   **Custom Variants**: If you select "Custom" for style/POV/length, the extension won't touch that part.
*   **What is Updated**:
    *   The core text of the standard Yablochny prompts (System prompt, Length instructions, etc.) is updated to the latest version.
*   **Fixing Bugs**: If the preset is completely broken, delete it using the trash icon in the Sliders tab and click **Sync preset** in the extension to regenerate it from scratch.

### 4. Fine-Tuning Formatting (Butterfly Tab)
Manage formatting, Story String, and Reasoning in the **Formatting** tab (3rd icon, Butterfly).
*   **Reasoning Block and Prefill**:
    *   **Gemini / Claude**: Set the **Start Reply With** (Prefill) field to `<think>`. This encourages these models to use their "reasoning" capabilities. Use **Auto-Parse Reasoning** to hide the thoughts in a neat collapsible block.
    *   **GPT-4 / o1**: Keep the **Start Reply With** field **COMPLETELY EMPTY**. Filling it can cause logic errors or halluncinations.
*   **Symbols in the Preset**:
    *   **‚óà (System)**: The technical core. Do not disable.
    *   **‚Üó / ‚Üò / ‚Üí (Option)**: Mutually exclusive options. Enable only one in a group.
    *   **‚îå / ‚îî (Connection)**: Linked prompts.
    *   **‚ó¶ (Personal)**: Optional modifiers for style and tone.
