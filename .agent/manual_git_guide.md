# Как обновлять файлы на GitHub вручную (для Windows)

Если ты не пользуешься VS Code или моими командами, вот как сделать всё самой.

## Способ 1: Через командную строку (Терминал)

1. **Зайди в папку** с пресетом через обычный проводник Windows.
2. В строке пути (сверху, где написано `C:\sillytavern\...`) нажми мышкой на пустое место, сотри всё, напиши **`cmd`** и нажми **Enter**.
   - Откроется черное окно (командная строка) прямо в этой папке.
3. По очереди вводи эти три команды (после каждой жми Enter):
   - `git add .`
   - `git commit -m "описание того что изменила"`
   - `git push origin main`

## Способ 2: Самый простой (GitHub Desktop)

Если не хочешь печатать команды, скачай программу **[GitHub Desktop](https://desktop.github.com/)**.

1. После установки нажми **File -> Add local repository** и выбери свою папку с пресетом.
2. Когда ты что-то изменишь в файлах, программа сама это увидит.
3. Внизу слева напиши название изменения и нажми синюю кнопку **Commit to main**.
4. Сверху нажми кнопку **Push origin**.
